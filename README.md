# awter-go

### Otter URL shortner
#### Ability to shorten with passwords and also get the QR code for the link

---

## Stack
- Go
- Chi
- Redis/MySQL

## Basic Functioning

Uses
nacl/secretbox to encrypt the main URL
scrypt to generate the key from the password
bcrypt to check the password
